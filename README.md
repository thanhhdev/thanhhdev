## Về repo này

Mình giữ repo này để:

- lưu lại các bài tập Java & Spring Boot
- thực hành Spring Data JPA, Thymeleaf, REST API
- giúp việc ôn lại kiến thức rõ ràng hơn
- chia sẻ code cho bạn bè và thầy cô

## Nội dung chính

- `TH1/` — Java cơ bản, Maven, project structure
- `Chuong2/` — Spring Core, dependency injection, bean lifecycle
- `Chuong3/` — Ứng dụng web Spring MVC + Thymeleaf
- `Lab4/`, `Lab5/` — Spring Data JPA, entity mapping, CRUD

## Những điều mình đang học

- Spring Boot application lifecycle
- Spring bean scope và DI
- Viết repository với Spring Data JPA
- Tạo REST API và xử lý form trong Spring MVC
- Đọc cấu hình `application.properties` và Spring profiles
